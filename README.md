# Usage
<!-- usage -->
```sh-session
$ npm install -g @awilmoth/unicli

$ unicli COMMAND

$ unicli (-v|--version|version)
unicli/1.0.0 darwin-x64 node-v12.19.0

$ unicli --help [COMMAND]
```
